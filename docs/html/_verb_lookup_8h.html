Detailed Description

Defines the VerbLookup class used for mapping text input to verb enums.

The VerbLookup class maintains a dictionary of valid player verbs and provides lookup, validation, and display functionality. It converts normalized player input strings into corresponding Verb enum values. Unknown or invalid inputs are handled gracefully as Verb::Unknown.
