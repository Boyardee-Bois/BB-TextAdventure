Detailed Description

Represents a single location in the game world.

Each Room contains:

  • A name and description.
  • A visited flag indicating whether the player has entered before.
  • A map of exits connecting it to neighboring rooms.

The Room class provides methods to retrieve information, manage exits, and print room details for player interaction.

Note
Rooms are typically created and linked during game initialization.
