Detailed Description

Represents a single interactable or collectible object in the game.

Each Item contains a name and a description that describe what the item is and how it can be interacted with. Items can be stored in rooms, inventories, or used for interactions within gameplay logic.

Note
Extend this class as needed to support item types, stats, or effects.
