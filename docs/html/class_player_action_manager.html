Detailed Description

Handles and executes player-related actions in the game.

Acts as the bridge between parsed Command objects and game state updates. Interprets player intent (movement, help, etc.) and applies effects to relevant game objects such as the Player and Map.

Current functionality includes movement and help display. Future expansions may include item handling, combat, and NPC interaction.

Note
This class executes actions but does not parse player input.
