Detailed Description

Represents the player and manages their current location and position.

The Player class stores a pointer to the current room the player is in, along with X and Y coordinates for map tracking. It provides methods to get and set these values, as well as move the player based on direction input.

Note
This class will be expanded to include features such as inventory, player stats, and combat.
