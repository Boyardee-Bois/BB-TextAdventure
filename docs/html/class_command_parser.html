Detailed Description

Converts player text input into Command objects.

Parses player input into structured Command objects.

Splits input on whitespace, lowercases tokens, and resolves them via VerbLookup and NounLookup. Does not execute gameplay.

The CommandParser processes text entered by the player, normalizes it (trimming and converting to lowercase), and interprets it into a structured Command object. It relies on the VerbLookup and NounLookup.

Command CommandParser::getCommand ( )

Reads, parses, and returns a structured Command object.

Read and parse one player command from stdin.

  • Reads one line from standard input.
  • Normalizes and tokenizes the line.
  • Maps the first token to a Verb and the second to a Noun.
  • Returns (Verb::Quit, Noun::Unknown) if input reading fails.
Returns
A Command object representing the player's input.
A Command with verb and noun, or (Verb::Quit, Noun::Unknown) on EOF.
